# SHA1 Password Checker

A simple educational Python project that demonstrates how **SHA-1 password cracking** works and how hashes can be compared with passwords from a wordlist.

⚠️ **This project is for educational purposes only.**

---

## Description

This program demonstrates the basic concept of password hashing using the **SHA-1 algorithm**.

The program works by:

1. Loading a **wordlist of passwords**
2. Give as input the SHA1 of the password 
4. Comparing the generated hash with the target hashes
5. Printing the matching password when a match is found
